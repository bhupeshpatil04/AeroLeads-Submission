# AeroLeads Assignment Repository

This repository contains all 3 tasks from the AeroLeads AI/Software Developer assignment.

## 1️⃣ LinkedIn Scraper
Scrapes 20 sample LinkedIn profiles using Selenium and saves them in CSV.

## 2️⃣ Autodialer App (Ruby on Rails)
Automatically calls numbers via Twilio and supports an AI prompt interface.

## 3️⃣ AI Blog Generator (Flask)
Generates programming blogs using AI APIs.
